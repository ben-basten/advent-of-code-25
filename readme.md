# Advent of Code 2025

Version control for the Advent of Code challenges.

<https://adventofcode.com/2025>

## Running a day

1. `cp .env.template .env`
2. Populate the `.env` file with the session token.
3. Navigate into the day directory
4. `node --import=tsx --env-file=../.env index.js`
